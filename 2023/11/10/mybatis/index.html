介绍

  • MyBatis是一款优秀的持久层框架，用于简化JDBC的开发

  • cotroller(控制层) -> ervice(业务层) -> dao(持久层) -> 数据库

MyBatis入门

快速入门

  1. 准备入门 （创建springboot工程、数据库表user、实体类User）

    1. File-New-Module

    2. 选择spring initializer，Language选择Java，Type选择maven，Location设定项目路径，Java选择Project SDK 17，Java选择17，点击Next

    3. Dependencies选择SQL中的MySQL Driver和MyBatis Framework，点击create

    4. 删除除了src目录和pom.xml以外的所有文件

    5. 如果要读取数据库的某个表User，就要创建数据库表对应的实体类User，实体类的属性和数据库表的字段属性相同或相似即可

  2. 引入MyBatis的相关依赖，配置mybatis（数据库连接信息）

    1. 配置main目录的resource目录下的application.properties文件（写入以下内容）

      # 配置数据库的连接信息 四要素
      # 驱动类名称
      spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
      # 数据库连接的url
      spring.datasource.url=jdbc:mysql://localhost:3306/mybatis
      # 数据库连接的用户名
      spring.datasource.username=root
      # 数据库连接的密码
      spring.datasource.password=123456

  3. 编写SQL语句（注解/XML)

    1. 创建目录mapper，在目录下创建UserMapper作为持久层（与数据库连接）的对象

    2. 创建目录pojo，在目录下创建User类作为数据的容器

    3. 在test目录下的测试类中编写测试用例

JDBC介绍

  • JDBC（Java DataBase Connectivity），就是使用Java语言操作关系型数据库的一套API。
  • JDBC本质是sun公司官方定义的一套操作所有关系型数据库的规范，即接口，具体的实现取决于各个数据库厂商（MySQL，ORACLE，SQL Server）提供的实现细节。
  • 我们可以使用这套接口（JDBC）编程，真正执行的代码是驱动包中的实现类。
